### 注意事项
- 沟通模式为飞书（`comm_mode: feishu`），开局先 `feishu_resume`
- 评估看板位于 `eval/` 目录，需要 HTTP 服务器运行：`cd eval && python -m http.server 8090`，然后访问 `http://localhost:8090`
- 看板使用 ES modules + Mermaid.js CDN，不支持直接 file:// 打开
- 人天列为空，需用户自行填写
- 状态列是下拉框：待评估 / 已确认 / 开发中 / 已完成 / 暂缓
- 技术方案分配可以打标签 + 给百分比
- 点击行打开右侧详情面板，所有字段可编辑
- 数据变更自动保存到 localStorage
- 方案设计 Tab 包含两套方案供客户选择
